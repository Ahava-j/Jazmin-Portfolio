Counting Inversions

Project 1

This project involved implementing an inversion count algorithm in C++ to practice designing efficient solutions for problems involving array comparisons. An inversion is a pair of elements in an array where the earlier element is greater than the later one (i.e., arr[i] > arr[j] for i < j). Counting these inversions is useful in various domains like measuring how far a sequence is from being sorted valuable in sorting analysis, competitive programming, and data preprocessing tasks. The main purpose of the project was to deepen my understanding of divide-and-conquer algorithms and improve my ability to write time efficient code. I used C++ and the Standard Template Library (STL) for vectors and basic I/O, and I worked in Visual Studio Code using the G++ compiler.
